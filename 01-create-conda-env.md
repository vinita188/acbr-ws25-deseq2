# create conda environment in miniconda
i am creating a conda environmnet with DESeq2 data analysis with R and Bioconductor in my system.


## code
Create conda environment
```{cmd}
conda create -n R4.5WS

```


Activate conda env 

```{cmd}
conda activate R4.5WS

```

Install R in environmnet 
```{cmd}
conda install conda-forge::r-base

```
